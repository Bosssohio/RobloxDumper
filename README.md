## What is RobloxDumper

RobloxDumper is just a dumper tool that can dump offsets and it runs roblox process and scans memory and steal offsets

## How to Run it?:

* You can always run RobloxDumper but require to join any game in Roblox

## How to setup

* 1: Open the requiredumper.rbxl
* 2: After It's launched, go to File -> Pubilsh
* 3: Make sure create the game for any label
* 4: If it's done for publish and set to Pubilc, go to your game and run RobloxDumper.exe



NOTE: Any game will work for stages except script stage, if its have test scripts, it will get offsets next time

WARNING: This includes stage and Script stage will find the test localscript and modulescript are generated by roblox in any game if its dont, it will skip and goes back and there is requiredumper.rbxl included have test script and also doesn't include cs, py, csv, more output saved (it will added in v1.5)
